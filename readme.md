## 📐 360° — أدوات الزوايا السعرية | الزمنية | ADAM KAMAL

منصة تحليلية متقدمة تعتمد على الخوارزميات الرياضية والهندسة السعرية وفق نظرية جان (Gann Theory). توفر المنصة أدوات دقيقة لحساب الزوايا السعرية، تحديد النماذج الهندسية، وقياس مناطق التوافق الرياضي (Clusters) بين نسب فيبوناتشي وزوايا جان. كما تتضمن معالجة لحظية لبيانات السوق المباشرة عبر بروتوكول WebSocket لاستخراج مستويات الارتكاز (Pivots)، مستويات كاماريلا، ومستويات موراي الرياضية بأسلوب عقلاني وموضوعي.
